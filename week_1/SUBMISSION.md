# Week 1 Submission

## Overview

I built a multi-turn chatbot in Python using OpenRouter and the OpenAI. The chatbot allows the user to choose from 3 free models, remembers previous messages by summarising old enough messages, and allows the user to have a smooth conversation across multiple turns.

## Features Implemented

* Multi-turn conversation loop (while loop)
* Conversation history using a messages list
* Model selection at startup amongst 3 free models
* Exit command (`exit` / `quit`)
* History reset command (`/reset`)
* Token usage command (`/tokens`)
* API key loading from `.env`
* `ChatAgent` class implementation
* Rolling buffer to limit the actual conversation messages stored in the list.
* Automatic conversation compaction using the model itself: compaction and summarize functions

## Design Decisions

The chatbot stores conversation history in a `messages` list containing system, user, and assistant messages. On every API call, the entire conversation history is sent to the model because LLM APIs are stateless and do not remember previous requests.

To prevent the history from growing infinitely, I implemented a rolling buffer. When the maximum number of turns is exceeded, older messages are compacted into a summary generated by the model. This summary is stored as a system message while recent messages are kept unchanged. This ensures the model understands the complete data of previous 2 turns while maintaining a summary of important facts of older messages.

I also allowed the user to choose between multiple OpenRouter models before starting the conversation.

## What I Learned

The most important thing I learned is that language models are stateless. Conversation memory is created by storing previous messages and resending them with each request.

I also learned how API responses are structured, how to use environment variables for API keys, how classes help organize code, and how context-window limitations can be managed through buffering and summarization.

I feel really happy to have created my very own chatbot, which I can configure as I want!
